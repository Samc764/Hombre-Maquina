# Que es un sistema
## Son un conjunto de elementos que cumplen una o mas funciones.

## Entradas

- Ambos son sistemas con entradas y salidas
Ser humano: recibe alimentos, agua, oxígeno (entradas) y produce energía, movimiento, calor, desechos (salidas).
Microondas: recibe electricidad y alimentos (entradas) y produce calor y comida calentada (salidas)

## Sistema de control

- Tienen un sistema de control
Ser humano: el cerebro y el sistema nervioso controlan todo.
Microondas: tiene un panel de control o circuito electrónico que regula tiempo y potencia.

## Energia

- Usan energía para funcionar
Ser humano: usa energía química (de los alimentos).
Microondas: usa energía eléctrica.

## Calor

- Generan y manejan calor
Ser humano: produce calor corporal (temperatura ~37°C).
Microondas: genera calor para calentar comida.

## Objetivo

- Tienen un objetivo o función
Ser humano: vivir, adaptarse, realizar actividades.
Microondas: calentar o cocinar alimentos.

## Seguridad

- Ambos tienen mecanismos de seguridad
Ser humano: el cuerpo tiene defensas como el sistema inmunológico, reflejos (retirar la mano del fuego) y regulación interna para evitar daños.
Microondas: tiene seguros como el bloqueo de la puerta (no funciona si está abierta) y sistemas que evitan sobrecalentamiento.

# Conclusión.

## Como pueden ver los humanos somos muy similares a las maquinas. Ambos tenemos sistemas en nuestro interior que funcionan de maneras similares
